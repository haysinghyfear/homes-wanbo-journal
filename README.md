# homes-wanbo-journal

这是一个用于归档和发布独立 HTML 页面的仓库。仓库内收录了多个不依赖特定域名的静态页面，主要用于个人记录、实验性展示或内容存档。

## 目录结构

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名通常反映其内容或用途。
- `assets/`：按需组织，与对应页面配合使用。

## 页面归档说明

本仓库中的页面均为独立 HTML 文件，不依赖外部服务器或数据库。每个页面可直接在浏览器中打开查看，适合作为轻量级内容存档。

- 所有页面均为静态内容，部分可能包含少量 JavaScript 用于交互或展示。
- 页面之间无强耦合，可单独引用或部署。
- 归档内容可能不定期更新或清理。

## 维护说明

- 本仓库由个人维护，主要用于备份和分享一些独立页面。
- 提交记录将反映页面新增、修改或移除的历史。
- 不保证所有页面长期可用或内容永久不变。
- 如果您发现任何页面存在明显问题（如链接失效、显示异常），欢迎提交 Issue 或 Pull Request。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/homes-wanbo-journal.git
   ```
2. 直接打开 `pages/` 下的 HTML 文件即可浏览。
3. 也可将 `pages/` 目录部署到任何静态服务器或 GitHub Pages 中。
